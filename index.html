Travian Defence Helper

How to Use

  1. Go to Village Statistics → Troops in the game.
    Path: /village/statistics/troops
  2. Press Ctrl + U to view the page source.
  3. Copy the entire code (Ctrl + A, then Ctrl + C).
  4. Select your Tribe below.
  5. Paste the code into the text area.
  6. Enter the Target URL (e.g. from the map) and optional Landing Time.
  7. Click Process Data.
